# akka-distributed-workers-scala
Akka Distributed Workers with Scala Example

## Running

### Requirements

- Java 8

### Local

- Enter `./sbt` on MacOS/Linux or `sbt.bat` on Windows to start sbt.
- And then, when console indicates *>*, execute `run`
